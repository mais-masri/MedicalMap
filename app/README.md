# 🩺 MediMap – Android Health Tracking Application

MediMap is an Android health-tracking mobile application designed to help users monitor important aspects of their daily wellness such as steps, hydration, meals, workouts, and walking routes.

The application integrates **Artificial Intelligence and Machine Learning** with Android development to provide smarter and more personalized health insights.

This project was developed as part of a **Group Software Engineering Project focused on AI integration with Android application development using Java.**

---

# 📱 Main Features

## 🏠 Home Page
The main dashboard provides quick access to the main features of the application:

- Step tracking
- Water intake tracking
- Meal planning
- Workout planning
- Walking trail map

This allows users to manage their daily health activities from one central page.

---

# 👣 Steps Tracker
The steps tracker helps users stay active by:

- Counting daily steps
- Showing progress toward a step goal
- Sending notifications when the goal is reached
- Displaying step history

This encourages users to stay physically active and track their improvement.

---

# 💧 Hydration Tracker

The hydration tracker allows users to monitor their daily water intake.

Features include:

- Logging water consumption
- Displaying daily hydration progress
- Notifications when the hydration goal is reached
- Viewing hydration history

Maintaining proper hydration is important for overall health.

---

# 🥗 Meal Tracking Plan

The meal tracking module allows users to manage their daily nutrition.

Users can:

- Track meals throughout the day
- View calories and nutritional values
- Access meal suggestions
- Manage weekly meal plans

This helps users maintain healthier eating habits.

---

# 🏋️ Workout Tracking

The workout tracker allows users to log and manage their exercise routines.

Users can:

- Log workouts
- Track workout types and duration
- Follow a weekly training plan
- View workout history

This helps maintain consistent fitness habits.

---

# 🗺 Map & Trail Tracking

The map feature allows users to explore and track walking trails.

Users can:

- View nearby walking trails
- Track routes
- Add new trail points

This encourages outdoor activity and exploration.

---

# 🤖 AI & Machine Learning Integration

One of the main goals of this project was to **integrate Artificial Intelligence into an Android application**.

The machine learning component analyzes user health data to provide smarter insights.

### AI Workflow

The AI component includes:

• Creating a dataset based on user health information and activity data.  
• Training a machine learning model using this dataset.  
• Identifying patterns between **meal plans, workouts, and health goals**.

The model learns relationships between:

- Nutrition and calorie intake
- Workout intensity
- User fitness goals

Using these patterns, the system can help generate **more personalized health suggestions**.

This demonstrates how **Machine Learning can be integrated with Android applications to improve user experience and provide intelligent recommendations.**
